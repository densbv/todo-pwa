# ToDo PWA

Простое офлайн PWA-приложение для списков дел, покупок и чек-листов.
Создавалось как лёгкий инструмент «не держать всё в голове».

Работает без аккаунтов, рекламы и сервера.

## Зачем

* быстро накидать список дел или покупок
* пользоваться с телефона как приложением
* не зависеть от интернета
* практика PWA, архитектуры и работы с состоянием

## Возможности

* несколько списков
* создание и удаление списков
* наименование списка (по умолчанию — дата и время)
* добавление и удаление задач
* отметка выполненных задач (тап по задаче)
* сортировка задач по дате
* локальное сохранение данных (localStorage)
* установка на телефон (PWA)
* работа офлайн

## Архитектура

* один экран со списками
* экран конкретного списка
* состояние хранится в localStorage
* offline-first PWA (Service Worker + Cache)

## Технологии

* Vanilla JavaScript
* HTML / CSS
* Bootstrap 
* PWA (Manifest + Service Worker)
